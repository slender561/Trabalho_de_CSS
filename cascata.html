Como funciona a Ordem da Cascata no CSS ?

Quando tiver varios estilos em um elemento como o CSS ira Seguir

  • Inline
  • Estilo incorporado ou um estilo externo...Nesse caso depende da ordem definida dentro da head
  • Estilo padrao do proprio Navegador


    • Portanto quando esta Inline ou seja diretamente na tag tera mais preferencia do que um arquivo de style externo ou interno na propria HEAD

    Quando temos dois elementos definidos de forma praticamente igual, vale qual tiver mais especificidade no CSS

    Quando uma Tag tive !important Ela tera prioridade sobre as demais formatacoes de CSS

    Exemplo: h1{!important color: blueviolet;}

    esta cor ira transpor todas as outras formatacoes de CSS
