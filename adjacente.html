Seletor Irmao Adjacente

Um seletor irmao adjacente é nada mais nada menos que dois ou mais seletores simples que sao separados por um sinal de + esse mesmo seletor usa de parametro o irmao adjacente do primeiro elemento...

A forma de usar e dentro da <style> voce coloca p+p{color: rgb(231, 38, 183); ou outras definicioes que voce quer colocar dentro dessa style}

<p> esse sera pulado</p>

<p>Esse sera pintado de rosa diferente dos outros que sera pintado de roxo</p>

Os Seletores funcionam quando determinado elemento tem um valor especifico no seletor

No total sao 4

[att] - Casa com qualquer elemento com o atributo att independente do seu valor

[att=val] - Casa com qualquer elemento com o atributo att cujo valor seja “val”

[att~=val] - Casa com valor de um atributo que pertence a uma lista de valores separados por espaço

[att|=val] - Casa com qualquer elemento que tenha um atributo att de valor igual a um valor qualquer separado por um hífen de um valor começando com “val”. O principal uso deste seletor é o de casar elementos com um valor de idioma especificado no atributo lang, por exemplo; “en-us”, “en-gb”, etc.

vamos para alguns Exemplos:

Nesse exemplo a regra sempre casara nos elemento h1 que tenham o atributo titulo

Exemplo: p[titulo]{color: blueviolet;}

No proximo exemplo o seletor ira casar com todos elementos de uma div com o mesmo atributo na classe

div[class=main] {color: blueviolet;}

Ja para atingir todos elementos em um 'td' com um atributo header de valor 'exe1' temos que usar

td[headers~=exe1]{color:blueviolet;}

no ultimo exemplo o seletor seguinte ira atingir o elemento p com todos os valores com atributo lang

p[lang|=pt] {color:blueviolet;}
