Universal

O seletor Universal, que para usá-lo utiliza-se o simobolo (*), ele aplica estilos a elementos de qualquer tipo, normalmente
utilizado para resetar (refazer) algumas propriedades que vem como pedrão nos navegadores

Exemplo:
* [lang^=pt] {
color: green;
}

*.atencao {
color: red;
}

*#conteudoprincipal {
border: 1px solid blue;
}

.flutuando {
float: left
}
